Menginstall Ubuntu

Akhirnya berhasil juga install Ubuntu. Sebenarnya ingin install Fedora Core tapi daripada nyari2 CD isntaller lagi, langsung saja pakai ubuntu, kan sudah punya. Dulu CD installer-nya pesen gratis dari https://shipit.ubuntu.com, pesen 10 CD sekaligus (untuk 3 jenis komputer berbeda).
