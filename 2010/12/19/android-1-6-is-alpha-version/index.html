Android issues on Xperia X8

#1 : Dec 19, 2010

Sekedar sharing, mungkin ada yang mengalami masalah seperti ini di se X8 (atau sejenis) ber-OS android

“the application home screen(process com.sonyericsson.homescreen) has stopped unexpectedly.please try again.”

solusi:

-download software updater di web se.

-lakukan langkah2 yang diberikan.

software ini nanti akan mendownload firmware terbaru untuk android yang anda miliki. cukup besar ~130 an MB. setelah download selesai firmware akan diprogramkan di handphone. pada waktu saya dapet masalah ini, sudah org yang banyak bertanya di forum se, namun jawabannya saya pikir kurang jelas. kenapa banyak org bertanya?, karena pesan error diatas menyebabkan hanphone crash a.k.a tidak bisa dipakai :p . Itu terjadi (untuk saya) , setelah install software ke handphone.

setelah update, semuanya jadi ok, dan ternyata tiba2 sekarang firmware saya  menjadi android 2.1 update 1 :).

semoga bermanfaat.

**#2  : March 12, 2011 **

Saya nemu bugs lagi!!. entah mulai nya kapan (kemungkinan besar sejak update ke 2.1 diatas), X8 setelah beberapa detik (sekitar 3-5 detikkan) akan restart ketika ditancapkan ke port USB sebagai flash disk. Awalnya saya menduga port usb saya bermasalah, namun setelah mencoba di beberapa komputer yang berbeda, hasilnya tetap sama. Masalah ini sangat mengganggu, karena kita tidak akan bisa mengakses SD card ponsel sebagai mass storage device. Bagaimana mengatasinya? . hmmm… ini sebenarnya bukan solusi terbaik , tapi solusi yang bisa dipakai.

  1. colok ke USB seperti biasa,

  2. ketika ponsel nge-prompt menanyakan mode (sbg usb,atau charge saja), pilih charge telepon.

  3. di menu atas (menu yang bisa digeser, di main page ponsel), pilih usb dan mount, dan sdcard sudah bisa diakses dan ponsel tidak restart.

Untuk benar2 menghilangkan bugs ini, sepertinya  harus update firmware lagi :p (lagi males)
